# yuque-to-local

End-to-end case for Yuque password source download, optional image transform,
and local filesystem deploy.

Required env:

- `ELOG_E2E_YUQUE_USERNAME`
- `ELOG_E2E_YUQUE_PWD`
- `ELOG_E2E_YUQUE_LOGIN`
- `ELOG_E2E_YUQUE_REPO`

Manual path and image settings live in `elog.config.ts` under `e2eProfile`.
