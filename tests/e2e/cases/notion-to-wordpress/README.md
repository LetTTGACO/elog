# notion-to-wordpress

End-to-end case for Notion source download and WordPress deploy.

Required env:

- `ELOG_E2E_NOTION_TOKEN`
- `ELOG_E2E_NOTION_DATA_SOURCE_ID`
- `ELOG_E2E_WORDPRESS_ENDPOINT`
- `ELOG_E2E_WORDPRESS_USERNAME`
- `ELOG_E2E_WORDPRESS_PASSWORD`

Manual image settings live in `elog.config.ts` under `e2eProfile`.
